# Bigbang
## 这是一款集求助任务发布接受和二手市场功能于一体的平台类APP，供交流展示使用。会持续更新完善，目前功能还未完全实现。
> 已经使用并且即将使用的框架有：<br>
网络：Retrofit+rxjava<br>
数据库：ObjectBox<br>
图片：Glide<br>
注入：Butter Knife<br>
事件总线：Eventbus<br>
地图SDK：高德地图SDK
